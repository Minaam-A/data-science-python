*Trainers*: 
- Samuel H. Lewis

*Structure*:
Week | w/c | Content | Materials 
:---:|:---:|---------|-----------

2 | 29th June 2020 | Basic Python concepts II | lecture , assignment , solution

4 | 13th July 2020 | Data handling | lecture , assignment , solution

*Format*:
